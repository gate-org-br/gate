data-filter

When the data-filter attribute is applied to an input field, it enables dynamic filtering functionality within its parent table's <tbody>. Users can input a value into the associated text input, and the table's body will be filtered based on the entered text. This feature is particularly useful for quickly narrowing down data in a table based on specific criteria.

In the provided example, the input field with the data-filter attribute is placed within the table's header, allowing users to filter the table based on values in the Name, Sex, and Age columns. As the user types, the table dynamically updates to display only the rows that match the entered text
