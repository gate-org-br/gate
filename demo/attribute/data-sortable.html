data-sortable

The data-sortable attribute is specifically designed for HTML tables, where it is applied to column headers to indicate that the content in that column is sortable. This attribute empowers users to interactively organize data within the table by clicking on the designated column headers. The sorting functionality is determined by the text content of the respective column. However, if a <td> element within a column includes the data-value attribute, the sorting is adjusted to use its the numeric value.
