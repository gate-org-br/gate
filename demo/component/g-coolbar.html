g-coolbar

The <g-coolbar> custom web component functions as a versatile toolbar, accommodating links and buttons while offering a visually appealing and responsive design. The component automatically styles elements based on their classes, defaulting to a secondary style. In situations where there is limited space to display all buttons, the component intelligently introduces scroll buttons. These scroll buttons facilitate user navigation, allowing them to access all content within the toolbar. You can use <hr> elements as greedy horizontal spacers and <br> elements as fixed-sized separators to enhance the layout and structure of the toolbar. The styling options include variations such as primary, alternative, tertiary, and danger, providing flexibility for customization.

Example 1

By default, the buttons within the <g-coolbar> component are aligned from right to left.

Example 2

By setting the reverse attribute, the alignment of the buttons is reversed, making them align from left to right.

Example 3

The disabled attribute in the <g-coolbar> component serves a unique purpose: when set, it hides all the elements within the toolbar and replaces them with an infinite progress bar. This behavior is designed to visually indicate to users that the toolbar is currently disabled or in a loading state, offering a clear and intuitive representation of the disabled state. The progress bar provides a visual cue of ongoing activity, ensuring users are aware of the current state of the toolbar while it's disabled.
