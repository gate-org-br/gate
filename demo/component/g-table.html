g-table

The g-table component extends the functionality of the standard HTML <table> by incorporating robust features for counting, filtering, and sorting data. Any child tag with a data-table-size attribute dynamically updates to display the total number of rows in the g-table body, ensuring users are always aware of the dataset's size. If the <table> body is empty, a custom message specified by the data-empty attribute is displayed, providing a clear indication when no records are found.

For dynamic filtering, include a child <input> with a data-filter attribute. This empowers users to input values, instantly triggering dynamic filtering within the <table> body. This feature is invaluable for rapidly refining data based on specific criteria, enhancing the user's ability to navigate and focus on relevant information.

To enable sorting, apply the data-sortable attribute to column headers. This indicates that the content in that column is sortable, allowing users to interactively organize data by clicking on designated column headers. By default, sorting is determined by the text content of the respective column. However, if a <td> element within a column includes the data-value attribute, the sorting is adjusted to use its numeric value, providing flexibility in organizing numerical data within the <table>.

These powerful functionalities collectively enhance the user experience, making g-table a versatile and feature-rich component for managing and presenting data effectively.
