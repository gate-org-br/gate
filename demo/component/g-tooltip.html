<g-tooltip>

The <g-tooltip> component provides a seamless way to attach tooltips to other elements within your HTML structure. To bind a tooltip, simply include a <g-tooltip> tag inside the target element, and within it, define the content you wish to display. This straightforward approach streamlines the process of creating and associating tooltips with specific elements, enhancing the overall user experience.

The <g-tooltip> allows you to precisely control its display position. By utilizing the optional position attribute, you can indicate the preferred location for the tooltip concerning its parent element. The engine will make every effort to display the tooltip at the specified position; however, if it's not feasible due to space constraints or other factors, the engine intelligently selects an alternative position for optimal visibility. This versatility ensures a seamless integration of tooltips while accommodating dynamic layout scenarios.

In addition to using the <g-tooltip> component, you have the alternative option of employing the data-tooltip and data-tooltip:position attributes on any HTML element. When utilizing data-tooltip, you can provide a JSON string as its value, defining the content you wish to display within the tooltip. The engine will automatically convert the JSON into HTML for an expressive and customizable tooltip presentation. Moreover, the optional data-tooltip:position attribute allows you to specify the preferred positioning of the tooltip.
