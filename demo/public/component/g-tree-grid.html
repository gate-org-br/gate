g-tree-grid

The g-tree-grid custom element is designed to enhance HTML tables by transforming them into interactive tree-like structures. Each row in the table represents a node in the tree, and hierarchical relationships are established based on the data-depth attribute. A collapsible icon in the first column allows users to expand or collapse nodes, revealing or hiding child nodes respectively. To use the g-tree-grid, include it in your HTML markup as a <table> element with the attribute is="g-tree-grid". Customize the content within the table body, using <tr> elements with the data-depth attribute to indicate hierarchy levels. The collapsible icons enable users to navigate and interact with the tree structure dynamically.
