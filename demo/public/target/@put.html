@post

Posts the result of the HTTP request to another supplied endpoint. If the endpoint is omitted, it defaults to the triggering element's action.

Example 1 –

On this example, the @form target fetches the form descriptor from the server using the HTTP get method and presents it's as a form to be edited by the user. Upon edition, the edited form is dispatched to the @post target. The @post target makes a HTTP post with the edited form to form.json and dispatches the result to the @value target. The @value target updates it's target value with the edited form.
