@template

Transforms JSON data into HTML using a specified Handlebars template. Before applying the transformation, the target adds helper functions (add, sub, mul and div) to Handlebars to allow basic arithmetic operations. Commented sections within the template are uncommented during this process. The resulting HTML is then passed to the next target in the pipeline.

Example 1 –

In this example, the @template target fetches json data from the server, converts it to HTML using the content of the #people template tag, and passes the transformed information to the @outer-html target. The @outer-html target replaces the <g-callout> with HTML data. Another @template trigger is invoked using the #person template tag when any row in the table is clicked passing the result to the @dialog trigger to show the person detail on a dialog.
