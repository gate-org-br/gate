Targets

The target indicate what to do with the result of the request. The default HTML targets start with the underline (_) character and by default only anchors and forms, using the target attribute, or buttons, using the formtarget attribute, can have targets. Gate adds some usefull target types starting with the at (@) character and provides the data-target attribute to allow targets to be specified on HTML elements other than forms, anchors and buttons.

Targets can have selectors as parameters. Selectors are strings that represent a series of navigation steps to locate a specific element within the DOM. It can start with this to indicate navigation relative to the trigger element or be a valid CSS selector to directly target an element on the document. Selectors can be quoted with double (") or single (') quotes. Steps can be separated by dots (.) or enclosed with square brackets ([]) and can be any of the following:

this
Skips the current step and continues to the next one.
parent()
Navigates to the parent of the element.
parent(number)
Navigates to the nth parent of the element.
parent(selector)
Navigates to the first parent of the element that matches the provided CSS selector.
prev()
Navigates to the previous sibling of the element.
prev(number)
Navigates to the nth previous sibling of the element.
prev(selector)
Navigates to the first previous sibling of the element that matches the provided CSS selector.
next()
Navigates to the next sibling of the element.
next(number)
Navigates to the nth next sibling of the element.
next(selector)
Navigates to the first next sibling of the element that matches the provided CSS selector.
search(selector)
Navigates to the first child element that matches the provided CSS selector.
[+number]
Navigates to the nth child element of the current source element.
[-number]
Navigates to the nth child element of the current source element in reverse order.
[selector]
Navigates to the first direct child element that matches the provided CSS selector.

The first example showcases that valid CSS selectors can be used to select a target element on the document. The following ones illustrate the usage of this to target elements on the DOM using the trigger element as reference.

section
Navigates to the next first section element of the document.
this.parent(section)
Navigates to the next first parent section element of the trigger element.
this.parent(2).next('.class-name')
Navigates to the next sibling with the class "class-name" of the third parent of the trigger element.
this.prev(-1)[3]
Navigates to the fourth child of the previous sibling of the trigger element.
