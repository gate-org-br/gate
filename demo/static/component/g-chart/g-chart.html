<g-chart>

Displays a dataset as a chart. The chart type (pie, polar, doughnut, line, column, bar, area, radar, polar) can be specified using the type attribute/property, defaulting to pie when omitted. The title attribute/property is the title of the chart to be displayed. The value attribute is the URL from where to fetch the value property. The value property structure is expected to be a two-dimensional JSON array, where the first row represents category headers and subsequent rows contain data corresponding to each category. The first column contains the names of the categories, while subsequent columns hold the values associated with each category.
