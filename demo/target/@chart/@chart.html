@chart

Displays a dataset fetched from an HTTP request as a chart within a dialog window. The chart type (pie, rose, donut, column, line, area, bar) can be specified as a parameter, defaulting to pie when omitted. The dataset structure is expected to be a two-dimensional array, where the first row represents category headers and subsequent rows contain data corresponding to each category. The first column contains the names of the categories, while subsequent columns hold the values associated with each category.
