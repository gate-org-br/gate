@dataset

The @dataset trigger extracts data from a <table> element and pass it to the pipeline. The table id can be passed on the data-action attribute or as a selector on the trigger parameter list.

Example 1さ

On this example, the @dataset trigger extracts data from the popularity <table> and pass it to the @value trigger. The @value trigger sets the value property of the <g-chart> element to be the data extracted from the table. This pattern facilitates the automatic generation of charts from tabular data.
