@debounce

Debouncing, a programming technique aimed at enhancing system performance, proves especially valuable in situations where resource-intensive tasks require regulated execution frequencies. In the context of user interfaces and event handling, the debounce mechanism strategically restricts the frequency of function calls within a defined time window. The @debounce target ensures that the subsequent target in the sequence is invoked only after the triggering event has ceased for the specified timeout duration. This helps prevent unnecessary or premature function calls. If the timeout parameter is omitted, it defaults to 1 second.
