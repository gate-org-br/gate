Display the result of the HTTP request in a dialog. The content will be loaded into a <div>; if the first parameter is set to fetch or into an <iframe> if it is omitted or set to frame. You can provide the width and height of the dialog as the second and third parameters, respectively. If the height is omitted, it defaults to the width. If both the width and height are omitted, the dialog will have 100% width and height with an 80-pixel margin when the screen width is greater than 640 pixels. If the trigger element or its parent contains a data-navigator attribute, the dialog will feature a navigator, enabling the user to navigate among all triggers located within it's parent element.
