@exec

The @exec target can be used to execute custom JavaScript code before passing the result of the request, without modification, to the next target in the pipeline. This target takes a single parameter: an arrow function. The arrow function is designed to receive the result of the request, and execute the custom JavaScript code.

Example 1 –

On this example, the @exec target fetches objects.json from the server and displays the first element on an alert box before dispathing the result to to the @json-to-html target. The @json-to-html target convert it to HTML and pass the converted information to the @outer-html target. The @outer-html target replaces the <g-callout> with HTML data.
