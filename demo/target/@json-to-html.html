@json-to-html

Formats JSON data into HTML and forwards it to the next trigger on the pipeline. JSON Objects are converted to <DL> tags, arrays to <UL> tags and all other JSON types to text.

On this example, the @json-to-html trigger fetches objects.json from the server, convert it to HTML and pass the converted information to the @outer-html trigger. The @outer-html trigger replaces the <g-callout> with HTML data.
