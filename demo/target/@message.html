Display the result of the request as a popup message. The type (success, error, warning, info) and duration (milliseconds) of the message can be passed as parameters in any order. If the type is omitted, it defaults to success. If the duration is omitted, the message sticks until closed by the user.

If the type parameter is omitted, it defaults to Success
