@populate

The @populate target is designed to populate a <datalist> or <select> tag with <option> tags generated from a JSON array obtained through a request. The first parameter is the selector of the target element; if omitted, defaults to the triggered element. The second parameter is the name of the attribute used to generate option values; if omitted, it defaults to "value". The third parameter is the name of the attribute used to generate option texts; if omitted, it defaults to "label".

Example 1 –

The @populate target can be used with action placeholders to populate a <select> tag based on the value of another one.

Example 2 –

The @populate target can be used with action placeholders and @trigger target to populate a <select> tag based on the values of any number of another ones.
