@report

Presents a dialog allowing the user to chose a report type from a list of available ones (PDF, XLS, ODT, and CSV). Upon selection, makes the request with the chosen type passed as parameter. The parameter name can be provided as the first parameter of the target, if omitted, it defaults to type.
