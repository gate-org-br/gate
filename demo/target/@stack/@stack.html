@stack

Present the outcome of an HTTP request within a self-contained frame, concealing underlying elements. The retrieved content dynamically populates a <div>, if the type parameter is omitted or set to fetch or an <iframe>, if it is set to frame. Notably, the frame possesses the capability to spawn additional frames, allowing the creation of a stack of nested frames.
