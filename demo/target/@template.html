@template

Transforms JSON data into HTML using a specified Handlebars template. Before applying the transformation, the target adds helper functions (add, sub, mul and div) to Handlebars to allow basic arithmetic operations. Commented sections within the template are uncommented during this process. The resulting HTML is then passed to the next target in the pipeline.
