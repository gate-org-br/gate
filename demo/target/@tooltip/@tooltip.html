@tooltip

Displays a tooltip containing the outcome of an HTTP request on the target element. JSON responses are seamlessly converted to HTML for optimal presentation. The position and size parameters can be provided in any order for enhanced flexibility. When a single size parameter is passed, the tooltip adopts equal width and height dimensions. If two size parameters are provided, the first parameter determines the width, and the second parameter determines the height of the tooltip. This versatile approach accommodates various preferences in specifying the appearance of the tooltip.
