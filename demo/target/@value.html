@value

Updates the value of the target element to the result of the HTTP request. You can pass the target element selector as a parameter or omit it to make the trigger element target itself.

Example 1 –

On this example, the @form target fetches the form.json file from the server using the HTTP get method and presents it's as a form to be edited by the user. Upon edition, the edited form is dispatched to the @post target. The @post target makes a HTTP post with the edited form to form.json and dispatches the result to the @value target. The @value target updates it's target value with the edited form.
