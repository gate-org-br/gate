The target indicate where to put the result of the request. The default HTML targets start with the underline (_) character and by default only anchors and forms, using the target attribute, or buttons, using the formtarget attribute, can have targets. Gate adds some usefull target types starting with the at (@) character and provides the data-target attribute to allow targets to be specified on HTML elements other than forms, anchors and buttons
